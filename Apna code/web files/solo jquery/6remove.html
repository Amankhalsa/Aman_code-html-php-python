1. Remove Elements
    2. We remove selected elements from the DOM using the remove() method. For example: HTML:,
    p style="color:red">Red p
    p style="color:green">Green p
    p style="color:blue">Blue p
    JS: $("p").eq(1).remove(); p
    Try It Yourself
  2. This removes Green, the second paragraph element. You can also use the remove() method on multiple selected elements, for example $("p").remove() removes all paragraphs. The jQuery remove() method removes the selected element(s), as well as its child elements.
