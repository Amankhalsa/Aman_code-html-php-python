waheguru ji ka khalsa

ਗੁਰਮੱਤ ਵਿੱਚ ਜਨਮ ਵੇਲੇ : ਸਰਬ ਲੋਹ ਦੇ ਵਰਤਨ ਵਿੱਚ 5 (ਜਰੂਰੀ) ਪਤਾਸੇ ਪਾਣੀ ਲੈ ਕੇ ਵੀਰ ਆਸਣ ਹੋ ਕੇ ਜਪੁਜੀ ਸਾਹਿਬ ਦਾ ਪਾਠ ਕਰਨਾ ਤੇ ਬੱਚੇ ਤੇ ਬੱਚੇ ਦੇ ਮਾਤਾ ਪਿਤਾ ਨੂੰ ਛਕਾਉਣ, ਜੇ ਮਾਤਾ ਪਿਤਾ ਸਿੱਖ ਨਾ ਹੋਣ ਤਾ ਨਈ ਦੇਣਾ !,

waheguru ji ki fateh

ਮੋਨੇ ਨੂੰ ਨਈ ਛਕਾਉਣ ਫੇਰ ਬੱਚੇ ਸਾਰਿਆ ਗੁਰਮੱਤ ਵਿਦਯਾ ਦੇਣੀ ਗੁਰਮੱਤ ਗ੍ਰੰਥ ਪੜ੍ਹਾਉਣੇ ਦਸਮ ਗ੍ਰੰਥ, ਸਰਬਲੋਹ ਗ੍ਰੰਥ ਤੇ ਹੋਰ ਸਾਰੀ ਸਿਖਿਆ ਦੇਣੀ,

ਤੇ ਫੇਰ ਵਿਆਹ ਵੇਲੇ ਪਹਿਲਾ ਅਰਦਾਸ ਕਰਨੀ ਮੰਗਨੀ ਰੋਕਾ ਜਾ ਕੁਜ ਹੋਰ ਸ਼ਬਦ ਵਿਹਾਰ ਨਈ ਕਰਨਾ ਫੇਰ 5 ਰੂਪੋ ਗੁਟਕਾ ਸਾਹਿਬ ਕੰਘਾ ਕੜਾ ਭੇਟ ਕਰਨਾ ਦੋਨਾ ਵਲੋਂ ਅਰਦਾਸ ਕਰਕੇ ਹੀ ਚਲਨਾ ਭਰੋਸਾ ਗੁਰੂ ਸਾਹਬ ਤੇ ਹੀ ਰਖਣਾ ਪੰਡਤਾ ਵਾਲੀ ਰੀਤ ਨਹੀ ਧਾਰਨੀ
