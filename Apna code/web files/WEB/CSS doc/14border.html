11.ਸਭ ਕਾਰਜਾ ਦੇ ਆਰੰਭ ਵੇਲੇ ਅਰਦਾਸ ਕਰਨੀ
12.ਜੰਮਨ, ਮਰਨ ਵਿਆਹ ਆਨੰਦ ਆਦਿ ਸਮੇ ਜਪੁਜੀ ਸਾਹਿਬ ਦਾ ਪਾਠ ਕਰਕੇ, 
 ਕੜਾਹ ਪ੍ਰਸਾਦਿ ਤਿਆਰ ਕਰਕੇ, ਆਨੰਦ ਸਾਹਿਬ ਦਾ ਪਾਠ,
 ਅਰਦਾਸ ਕਰਕੇ ਪੰਜਾ ਪਿਆਰਿਆ ਤੇ ਹਜੂਰੀ 
 ਗ੍ਰੰਥੀ ਸਿੰਘਾ ਦਾ ਵਰਤਾਰਾ ਵਰਤਾ ਕੇ ਰੱਖ ਉਪਰੰਤ ਸੰਗਤਾ ਨੂੰ ਵਰਤਾ ਦੇਣਾ
13.ਜਦ ਤੱਕ ਕੜਾਹ ਪ੍ਰਸਾਦ ਵਰਤਦਾ ਰਹੇ ਸਾਰੀ ਸੰਗਤ ਅਡੋਲ ਬੈਠੀ ਰਹੇ

18.ਰਹਿਤਵਾਨ ਤੇ ਨਾਮ ਜਪਣ ਵਾਲਿਆ ਗੁਰਸਿੱਖਾ ਦੀ ਸੰਗਤ ਕਰਨੀ
19.ਜਿਤਨੇ ਕਰਮ ਆਪਣੇ ਕਰਨ ਦੇ ਹੋਣ , ਓਹਨਾ ਦੇ ਕਰਨ ਵਿਚ ਆਲਸ ਨਹੀ ਕਰਨੀ
